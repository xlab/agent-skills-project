Get Started in Minutes

From zero to sharing your own resources in three simple steps.

1

Try an existing skill

No installation needed. Just run this command to add your first skill:

uvx add-skill kasperjunge/hello-world
2

Create your own repo

Scaffold your personal agent-resources repository with one command:

uvx create-agent-resources-repo --github
3

Share with the community

Push to GitHub and share your resources. Others can install with:

uvx add-skill your-username/your-skill
